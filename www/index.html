Welcome to Webserv

Project Overview

Webserv is a custom HTTP server written in C++.

Usage

To get started with Webserv, follow these steps:

  1. Downeload the repo
    2. https://github.com/MariiaZhytnikova/Webserv
  2. Compile the server:
    3. make
  3. Run the server with a configuration file:
    4. ./webServ [configuration file]
