Welcome to Webserv

A lightweight, C++ HTTP server implementing the mandatory and bonus parts of the 42 webserv project: virtual hosts, CGI, file uploads, custom error pages and more.

View repo

Quick test commands

curl -i http://localhost:8080/
curl -i -X POST -d "hello world" http://localhost:8080/uploads/echo
curl -v -F "file=tmp-test.txt" http://localhost:8080/uploads/ 
curl -i -X DELETE http://localhost:8080/resource
curl -i -F "file=@test.txt" http://localhost:8080/upload
printf "METHOD / HTTP/1.1\r\n Header1:value \r\n Header2:value \r\n\r\n body" | nc localhost 8080
printf "POST / HTTP/1.1\r\nHost: localhost:8080\r\nContent-Length: 9\r\n\r\nhello=123" | nc localhost 8080
printf "KILL / HTTP/1.1\r\nHost: localhost:8080\r\nContent-Length: 9\r\nConnection: close\r\n\r\nhello=123" | nc localhost 8080
siege -b http://localhost:8080/pages/ok.html
Replace port/path as configured in your `default.conf`
